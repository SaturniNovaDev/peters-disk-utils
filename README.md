# Peter's Disk Utils

Python-based suite of disk management tools.

The suite consists of two primary scripts: `disk-nls.py` for data acquisition and `disk-nav.py` for data inspection.

## Technical Specifications

- **Language:** Python 3.x
- **Core Dependencies:** `matplotlib`, `tqdm`
- **I/O Strategy:** `os.scandir` (Recursive iteration)
- **Data Format:** JSON (Nested Tree Structure)
- **Visual Output:** PDF, PNG, or JPG (Matplotlib backend)

## Component 1: Disk Analyzer (disk-nls.py)

This script performs a recursive traversal of the specified filesystem. It utilizes `os.scandir` to minimize system calls and improve performance on NTFS and ext4 filesystems.

### Parameters

| Argument   | Requirement | Description                                                        |
| :--------- | :---------- | :----------------------------------------------------------------- |
| `--target` | Required    | The drive letter (e.g., "C:") or absolute path to index.           |
| `--output` | Optional    | Path/filename for the visual ring chart (Default: `analysis.pdf`). |

### Functionality

1. **Indexing:** Traverses the entire directory tree starting from the target.
2. **Progress Tracking:** Displays an active item count via a green progress bar.
3. **Data Persistence:** Exports the full hierarchy to `disk_index.json` including:
   - Absolute root path
   - Indexation timestamp
   - Total size in bytes
   - Hierarchical node data (Name, Size, Type)
4. **Visualization:** Generates a ring (donut) chart of the top-level distribution, grouping items below a 1% threshold into an "Others" category.

## Component 2: Index Navigator (disk-nav.py)

A Command Line Interface (CLI) tool designed to explore the generated JSON index without requiring further disk access.

### Parameters

| Argument   | Requirement | Description                                                   |
| :--------- | :---------- | :------------------------------------------------------------ |
| `--source` | Required    | Path to the `disk_index.json` file generated by the analyzer. |

### Command Set

The navigator operates in an interactive REPL mode with the following commands:

- `ls`: Lists current directory contents, sorted by size (Descending).
- `cd <directory_name>`: Changes context to a specific child directory.
- `cd ..`: Returns to the parent directory.
- `exit` or `quit`: Terminates the session.

### Features

- **Automatic Scaling:** Sizes are dynamically converted from bytes to KB, MB, GB, or TB.
- **Percentage Calculation:** Calculates real-time disk occupancy relative to the parent node.
- **State Management:** Maintains a path stack to allow navigation through deeply nested structures.

## Installation and Execution

1. **Environment Setup:**

   ```bash
   pip install matplotlib tqdm
   ```

2. **Generate Index:**

   ```bash
   python disk-nls.py --target "C:" --output "storage_report.pdf"
   ```

3. **Navigate Output Index:**

   ```bash
   python disk-nav.py --source "disk_index.json"
   ```

## Notes

**Recursion:** The tool utilizes depth-first search for size accumulation.

**Permissions:** Some system directories may be inaccessible. These are caught as PermissionError and skipped to prevent script termination. Running the terminal as Administrator is recommended for full drive indexing.
